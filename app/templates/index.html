Pool Library

Pool switching note. The Avalon Q applies a pool change on next reboot. To make a pool switch take effect right away, enable reboot after pool switch on the rule (or check the box when manually applying). Reboots forfeit any in-flight shares — keep that in mind for high-priority work.

Schedule

How rules work. Each rule defines an action (workmode, soft-off, pool switch, etc.) for a specific time window on selected days. When the window ends, control passes to whatever rule covers the next moment, or to the default action if no rule matches. Rules don't auto-revert — you need a follow-up rule (or default action) to bring the miner back. Priority breaks ties when two rules overlap: higher number wins. Use it for "exception" rules like APS on-peak that should trump a general "always run Super" rule.

Manual Control
